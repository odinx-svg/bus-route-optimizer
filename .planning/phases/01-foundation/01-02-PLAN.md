    2. Framer Motion variants:
       ```typescript
       const buttonVariants: Variants = {
         initial: { scale: 1 },
         hover: {

           transition: { type: 'spring', stiffness: 400, damping: 10 }
         },
         tap: { scale: 0.98 }
       };
       ```

    3. Color classes by variant:
       - green: `bg-neon-green/10 border-neon-green text-neon-green hover:shadow-neon-green`
       - cyan: `bg-cyan-blue/10 border-cyan-blue text-cyan-blue hover:shadow-neon-cyan`

    4. Use `motion.button` from framer-motion with:
       - variants={buttonVariants}
       - initial="initial"
       - whileHover={disabled ? undefined : "hover"}
       - whileTap={disabled ? undefined : "tap"}

    5. Base classes: `px-6 py-3 rounded-lg border font-semibold transition-shadow duration-300`

    6. Disabled state: `opacity-50 cursor-not-allowed`

    Reference pattern from 01-RESEARCH.md "NeonButton Component" section.

    WCAG note: Neon green (#39FF14) on dark-bg (#0a0a0f) has 11.5:1 contrast ratio - PASSES AA for large text (buttons are 14px+ bold).
